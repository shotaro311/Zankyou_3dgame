# AGENTS.md

## Project Rules

- このプロジェクトは、GitHub Pagesで公開する単体HTMLゲームとして扱う。
- 公開対象は原則として`index.html`とドキュメントだけにする。
- 新しい秘密情報、APIキー、個人情報、`.env`は追加しない。
- ゲーム本体を変更した場合は、最低限`index.html`内のJavaScript構文チェックとブラウザ表示確認を行う。
- UI文言は日本語を主とし、作品タイトルや説明に英語を使う場合はポートフォリオで伝わる表現にする。

## File Layout

- `index.html`: ゲーム本体
- `README.md`: ポートフォリオ向け説明
- `PROJECT.md`: プロジェクト情報
- `progress/`: 作業進捗ログ
